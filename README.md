# My Java Learning Journey

Welcome to my Java learning repository! This project tracks my progress as I learn Java from scratch. It is built using Gradle.

### 1. Java Basics
- Syntax and Structure
- Variables and Data Types (Primitives and Objects)
- Operators (Arithmetic, Relational, Logical)

### 2. Control Flow
- Conditional Statements (`if`, `else if`, `else`, `switch`)
- Loops (`for`, `while`, `do-while`)

### 3. Object-Oriented Programming (OOP)
- Classes and Objects
- Methods and Constructors
- Inheritance, Polymorphism, Encapsulation, and Abstraction

### 4. Data Structures (In Progress)
- Arrays and 2D Arrays
- Collections Framework overview
- Lists (`ArrayList`, `LinkedList`)
- Sets and Maps (Currently exploring)

## Building and Running the Code

This project uses the Gradle build system. You can build and run the code using the Gradle wrapper included in the repository.

### To build the project:
```bash
./gradlew build
```

### To run the application:
```bash
./gradlew run
```
